# Скрипты проекта

Эта папка содержит вспомогательные утилиты и скрипты для проекта.

## ensure_init_files.py

Утилита для автоматического создания `__init__.py` файлов во всех Python пакетах проекта.

### Использование

**Автоматически:**
- Скрипт запускается автоматически через git hook при каждом коммите (`.git/hooks/pre-commit`)

**Вручную:**
```bash
# Из корня проекта
python scripts/ensure_init_files.py

# Или из папки scripts
python ensure_init_files.py ..
```

**Через bat-файл (Windows):**
```cmd
scripts\ensure_init.bat
```

### Что делает

- Рекурсивно обходит все папки проекта
- Находит папки с Python файлами (`.py`)
- Создает `__init__.py` файлы там, где их нет
- Игнорирует служебные папки: `venv`, `.git`, `__pycache__`, `scripts`, и др.
